Accessibility

Computing promotes accessibility by providing digital aids to those with additional needs. This includes:

  • Screen reading technology, which allows those with limited sight to have file content read aloud to them. This relies on correct use of alternative text for elements like images.
  • Voice recognition technology, which allows users to communicate with computers using speech commands.
  • Speciliased input and output devices including 'Sip and puff' switches, where users can enter information with their breath, or braille printers.

These technologies enabled by commputers make computers themselves accessible, and so promote the advantages of modern technology to a wide audience.
